# Funzionalità
## Tab Chiudi quella porta
- All'avvio parte in automatico una volta
- Parte cliccando sul **bottone** oppure con **barra spaziatrice** o **invio**

## Tab Tutti gli auDIO
- Lista filtrabile con la casella di testo in alto
- **Ctrl+F** per focussare la casella
- Filtro attivo mentre si digita
- Premere **invio** per riprodurre il file audio più in alto nella
- **Ctrl+(tasto del numpad)** per riprodurre il file i-esimo nella lista
(ad esempio pigiando Ctrl+3 viene riprodotto il terzo file. Se il valore
pigiato è maggiore del numero di elementi in lista viene riprodotto l'ultimo disponibile)
